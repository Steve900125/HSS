# Applications Demo

1. Install requriement lib
```
pip install -r requirements.txt
```
2. Run server "client_main.py", "server_a.py", "server_b.py"
```
python client_main.py
```
```
python server_a.py
```
```
python server_b.py
```
3. Send request (put in browser)
```
http://localhost:8300/send_keywords
```
